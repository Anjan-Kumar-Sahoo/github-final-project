# Django Admin Site Screenshot Instructions

## To capture the admin site screenshot (Task 3):

1. Run the Django server: `python manage.py runserver`
2. Navigate to: `http://localhost:8000/admin`
3. Login with your superuser credentials
4. Take a screenshot showing:
   - Authentication and Authorization section (Users, Groups)
   - OnlineCourse section (Courses, Lessons, Questions, Choices, Submissions, Instructors, Learners)
5. Save the screenshot as: **03-admin-site.png**

## To capture the final exam result screenshot (Task 7):

1. Make sure you have:
   - Created a course with lessons and questions in admin
   - Enrolled in the course
   - Submitted an exam with correct answers
2. Navigate to the result page after submitting
3. Take a screenshot showing:
   - "Congratulations" message at the top
   - Your score (e.g., "Your Score: 15/15")
   - Percentage (should be 80% or higher)
   - Detailed question-by-question results with checkmarks
4. Save the screenshot as: **07-final.png**

Place both screenshots in the root directory of your project before pushing to GitHub.
