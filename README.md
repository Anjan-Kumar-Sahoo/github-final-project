# GitHub Final Project

## Project Name: Simple Interest Calculator

This project is a simple interest calculator implementation created as part of the IBM Developer Skills Network course on Introduction to Git and GitHub.

## Description

A command-line tool that calculates simple interest based on user-provided principal amount, rate of interest, and time period.

## Features

- Calculate simple interest using the formula: Simple Interest = (Principal × Rate × Time) / 100
- User-friendly command-line interface
- Bash script implementation

## Getting Started

To use the simple interest calculator, run the `simple-interest.sh` script and follow the prompts.
